## Описание программы

Это графическое приложение - генератор надежных случайных паролей. При создании пароля можно выбирать длину (от 4 до 64 символов). А также выбирать тип символов, которые будут включены в случайную генерацию (доступны заглавные и строчные буквы, цифры и специальные символы).
Программа сохраняет историю и даты генерации паролей в файл "password_generator.json". И отображает историю прошлых генераций в окне.

## Как пользоваться программой

1. Запустить приложение.
2. Отметить чек-боксы с нужными символами
3. Ползунком выбрать желаемую длину пароля.
4. Нажать кнопку "Сгенерировать пароль".
5. Внизу отобразиться сгенерированный согласно заданным настройкам пароль.
6. Созданный пароль автоматически сохраниться в файле с историей "password_generator.json". В консоле отобразиться запись об успешном сохранении данных в файл. 
7. При отсутствии файла "password_generator.json" он будет создан автоматически.
8. Содержимое файла истории (от ранних дат к поздним) отобразиться в нижней части главного окна программы. Данные можно промотать с помощью скролбара.
